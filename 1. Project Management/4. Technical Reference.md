# Technical Reference

-----

## Tech Stack

| Layer | Technology | Version / Notes |
|-------|-----------|-----------------|
| Language | Swift | 5.9+ |
| UI Framework | SwiftUI | iOS 16+ (NavigationStack, async/await) |
| Architecture | MVVM + Combine | ViewModels own state + business logic; Views are declarative |
| Auth | Firebase Auth SDK | Sign in with Apple (mandatory), Google, email/password |
| Database | Firestore | Document model; offline persistence enabled; free Spark plan for MVP |
| Map | Mapbox Maps SDK for iOS | Satellite imagery, custom overlay layers, SwiftUI UIViewRepresentable wrapper |
| Course Data | Golfbert API | REST/JSON; GeoJSON hole geometry; cached in Firestore after first fetch |
| Subscriptions | StoreKit 2 + RevenueCat (recommended) | StoreKit 2 native async/await; RevenueCat for entitlement management + analytics |
| Location | CoreLocation | `whenInUse` permission; LocationService wrapper; GPS auto-detect in round mode |
| Local State | UserDefaults | Lightweight persistent state: distance unit, skew, onboarding step |
| Structured Cache | Firestore offline | Doubles as local cache for courses/holes — avoids CoreData complexity for MVP |
| Network Monitoring | NWPathMonitor | Offline indicator in nav bar |
| Testing | XCTest | Unit tests for DispersionEngine + BagViewModel unlock logic |

## Architecture

```
App Entry Point
  └── AppState (ObservableObject singleton)
        ├── AuthState: .unauthenticated | .onboarding | .authenticated
        └── CurrentUser + navigation state

Root View (switches on AuthState)
  ├── AuthFlow (NavigationStack)
  │     ├── SignInView
  │     ├── SignUpView
  │     └── ForgotPasswordView
  ├── OnboardingFlow (NavigationStack, forward-only)
  │     └── OnboardingViewModel (step enum, UserDefaults persistence)
  └── MainTabView
        ├── HomeTab → HomeView
        ├── RoundStudyTab → CourseSearchView(mode:)
        │     └── HoleSelectorView → HoleMapView
        ├── BagTab → BagManagementView → ClubDetailView
        └── ProfileTab → ProfileView

Key ViewModels:
  MapViewModel — shotCoordinate (Published), dispersion calc, distance calc, Combine chain
  BagViewModel — clubs array, hasAnyClubs / hasAllDistances / hasAllDispersions
  OnboardingViewModel — step enum, UserDefaults persistence
  CourseSearchViewModel — nearby courses, search, loading/error/empty state
  AuthViewModel — wraps AuthService, post-auth routing logic

Services (injected, not instantiated in ViewModels):
  AuthService — Firebase Auth wrappers (async)
  FirestoreService — generic read/write with Codable support
  NetworkService — URLSession wrapper for Golfbert (error handling, retry)
  LocationService — CoreLocation wrapper, publishes CLLocation

DispersionEngine (struct, pure functions — no services, fully testable):
  calculateWidth(distance:handicap:) → Double
  interpolateWidth(distance:clubData:) → Double
  applySkew(width:skew:) → (leftWidth, rightWidth)
  buildEllipse(center:aimDirection:leftWidth:rightWidth:length:) → [CLLocationCoordinate2D]
```

## Conventions

- **Naming**: Swift API Design Guidelines — `camelCase` for vars/funcs, `PascalCase` for types
- **Async**: `async/await` throughout; no callbacks or completion handlers
- **State**: `@Published` in ViewModels, `@StateObject` / `@EnvironmentObject` in Views
- **Errors**: typed `Error` enums per service; never `try!` in production code
- **Models**: all `Codable` with `CodingKeys` where Firestore field names differ from Swift names
- **Feature flags**: none for MVP — just change the code
- **Comments**: only where the WHY is non-obvious (workarounds, invariants)

## Environment & Setup

1. **Xcode** 15+ required (SwiftUI, Swift 5.9, StoreKit 2)
2. Create Xcode project: `GolfApp`, SwiftUI lifecycle, iOS 16 minimum
3. Add SPM dependencies (see above)
4. Create two Firebase projects: `golfapp-dev` and `golfapp-prod`
5. Download `GoogleService-Info.plist` for each; add to Xcode with build scheme targeting
6. Set Mapbox token in Info.plist: `MBXAccessToken`
7. Set Golfbert API key in Constants.swift (never commit key — use Xcode environment variable or xcconfig)
8. `.gitignore` already excludes `GoogleService-Info.plist` and `*.xcconfig` containing secrets

## Dispersion Formula (Tier 1)

```
dispersionWidth = (shotDistanceYards × 0.10) × handicapMultiplier

Handicap → Multiplier:
  0 (scratch)  → 0.6×
  5            → 0.8×
  10           → 1.0× (baseline)
  15           → 1.2×
  20           → 1.4×
  25           → 1.6×
  30+          → 1.8–2.0×

Handicap derived from averageScore using standard USGA approximation:
  handicap ≈ (averageScore - 72) × 0.96   (simplified; full calculation in HandicapCalculator.swift)
```

## Golfbert API

- Base URL: `https://api.golfbert.com/v1`
- Auth: API key in `Authorization: Bearer <key>` header
- Key endpoints: `GET /courses?lat=&lng=&radius=`, `GET /courses?search=`, `GET /courses/{id}`, `GET /courses/{id}/holes/{n}`
- Cache policy: write all course + hole data to Firestore on first fetch; check cache before calling; stale after 30 days
- Response format: JSON with GeoJSON geometry for fairways and greens

## Known Constraints / Gotchas

- **Mapbox UIViewRepresentable**: coordinator pattern needed for gesture delegation; can't use pure SwiftUI view
- **Sign in with Apple**: `ASAuthorizationControllerDelegate` must be on a `UIViewController` — use a helper coordinator
- **Firebase + Mapbox SPM**: both are large; first Xcode resolve takes ~5 min; DerivedData cache speeds subsequent builds
- **Golfbert coverage**: strong for US courses; international coverage may be sparse — note in onboarding if course not found
- **StoreKit 2 sandbox**: must test in-app purchases via TestFlight sandbox; Simulator StoreKit works for UI flow only
- **dispersionWidth floor**: 5 yards minimum regardless of formula output
- **dispersionWidth ceiling**: 50 yards maximum (or driver dispersion, whichever is lower)
- **Post-MVP fields**: `shot_coordinate`, `club_used`, `actual_distance`, `timestamp`, `gps_validated_distance`, `sample_count`, `total_rounds`, `total_shots` — added to models now but nothing writes to them until Tier 4
