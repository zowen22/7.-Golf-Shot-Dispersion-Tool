# Project Overview

*Generated from Problem Statement on: 2026-06-15*

-----

## Summary

An iOS caddie app that renders personalized shot dispersion ellipses on a live satellite hole map. Users drag a shot marker; the app shows a realistic outcome zone based on their handicap and club data. The core loop — drag, see dispersion, make smarter decision — works from day one without any club data and gets sharper as the user builds out their bag.

## Scope

### In:
- SwiftUI iOS app (iOS 16+, iPhone)
- 12-screen onboarding with paywall (StoreKit 2)
- Firebase Auth (Apple, Google, email/password)
- Hole map screen: Mapbox satellite, draggable shot marker, live dispersion ellipse, skew selector
- Tiered dispersion system (Tiers 1–3 for MVP)
- Bag management (club distances + optional dispersion widths)
- Course search: GPS nearby (round mode) + manual search (study mode)
- Hole selector with pre-fetch of all 18 holes
- Home screen, Profile/Settings screen
- Golfbert API integration with Firestore caching
- Firestore database (users, profiles, bags, clubs, courses, holes, rounds skeleton)
- Offline handling via Firestore persistence + NWPathMonitor indicator

### Out (Post-MVP):
- Scorecard and round history
- GPS shot tracking (Tier 4 dispersion auto-refinement)
- Wind and elevation adjustment
- Hazard avoidance / layup suggestions
- Social features and leaderboards
- Multiple bag profiles
- Arccos / Shot Scope import
- iPad support
